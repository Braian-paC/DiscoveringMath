Exponenciais

1. O que é Função Exponencial?

Uma função exponencial é aquela em que a variável aparece no expoente, como em y = a·b^x. Esse tipo de função descreve fenômenos em que a mudança depende do valor atual da grandeza.

Por exemplo, se uma população dobra a cada ano, o número de habitantes não cresce de forma linear, mas sim multiplicando-se por 2 a cada período. Isso é o comportamento típico da função exponencial. O valor inicial é representado por a, e a base b indica o fator de crescimento ou decaimento.

Quando b > 1, a função cresce rapidamente. Quando 0 < b < 1, ela decresce, aproximando-se de zero ao longo do tempo. Assim, a função exponencial é ideal para representar crescimento acelerado ou queda progressiva.

Exemplo: f(x) = 3·2^x. Para x = 0, temos f(0) = 3; para x = 1, f(1) = 6; para x = 2, f(2) = 12. O valor dobra a cada passo.

2. Para que serve?

As funções exponenciais modelam crescimento e decaimento em fenômenos reais, como população, juros compostos, desintegração radioativa e transmissão de vírus. Elas descrevem situações em que a mudança é proporcional ao valor atual.

Uma função típica é A(t) = A_0 · b^t. Se b > 1, o valor cresce; se 0 < b < 1, ele decresce. Essa estrutura permite prever como uma grandeza muda ao longo do tempo, muitas vezes de forma muito rápida.

Por isso, esse modelo é essencial para entender fenômenos que aumentam ou diminuem em ritmo acelerado, como investimentos, epidemias e decaimento de substâncias.

3. Aplicações reais

Na economia, ajudam a calcular juros compostos. Na biologia, descrevem crescimento populacional. Na física, modelam a decadência de substâncias radioativas. Em tecnologia, aparecem em cargas, redes e crescimento de dados.

Exemplo: uma população que dobra a cada ano pode ser descrita por P(t) = P_0 · 2^t, mostrando o crescimento exponencial ao longo do tempo. Se o valor inicial é 500 e o tempo é 3 anos, então P(3) = 500 · 2^3 = 4000.

Exemplos do cotidiano: 1) prever juros compostos; 2) modelar crescimento populacional; 3) analisar queda de bateria; 4) entender o espalhamento viral de uma campanha.

Exemplos específicos do cotidiano

1. Juros compostos: se você aplica R$ 500 com crescimento de 2% ao mês, a função é A(t) = 500(1,02)^t. Após 6 meses, A(6) ≈ 563,41.

2. População: se uma população dobra a cada ano, então P(t) = P_0 2^t. Com P_0 = 300, em 4 anos P(4) = 300 × 16 = 4800.

3. Bateria: se a carga cai 10% por hora, a função pode ser B(t) = 100(0,9)^t. Após 3 horas, B(3) ≈ 72,9%.

4. Vírus ou campanha viral: se cada pessoa compartilha com 3 novas pessoas, o crescimento é N(t) = N_0 3^t.
